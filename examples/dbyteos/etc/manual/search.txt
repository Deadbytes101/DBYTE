DByteOS Search Command

The search command provides deterministic scanning of workspace resources, project notes/tasks, journals, and agendas.

Usage:
  workspace search <text>
  project search <project> <text>
  task search <project> <text>
  daily search <text>
  search index <text>
  search status
  search rebuild
  search doctor
  search clear-cache
  search reset-demo

Examples:
  workspace search tests
  project search demo note
  task search demo tests
  daily search todo
  search index seed
  search status
  search rebuild
  search doctor
  search clear-cache
