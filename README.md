# DByte

A low-level scripting language designed for binary patching and memory manipulation.

## Features

- **Project Workflow**: Manage multiple files and modules with `Dbyte.toml`.
- **Dual Execution Engines**: Fast Tree-walk Interpreter and Bytecode VM.
- **Low-level Types**: Native support for `bytes` and mutable `buffer`.
- **Binary Manipulation**: Endian-aware reads and writes (`u8`, `u16_le`, `u32_be`, etc.).
- **File Patching**: Built-in `load`, `save`, `find`, and `replace` for binary files.
- **Robust Stdlib**: `math`, `fs`, `encoding`, `hash`, `env`, `buffer`, and `binary`.

## Quick Start

```powershell
# Create a new project
dbyte new my_patcher
cd my_patcher

# Run the project
dbyte run

# Run with VM engine
dbyte run --vm
```

## Example: File Patching

```dbyte
import std.buffer as buf

let b = buf.load("target.bin")
let pos = buf.find(b, b"\xDE\xAD\xBE\xEF")

if pos >= 0:
    buf.replace(b, pos, b"\x90\x90\x90\x90")
    buf.save("patched.bin", b)
```

## Commands

- `dbyte run <file>`: Execute a DByte script.
- `dbyte check <file>`: Perform static type checking.
- `dbyte test`: Run the test suite.
- `dbyte disasm <file>`: View generated bytecode.
- `dbyte --version`: Show version information.
