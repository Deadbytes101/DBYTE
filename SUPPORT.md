# Support

DBYTE is a working snapshot and may change quickly.

## Best Places to Start

- `README.md` for the project map.
- `LANGUAGE_SPEC.md` and `docs/language-spec.md` for language notes.
- `docs/` for DByteOS and Kernel Lab documents.
- `kernel-lab/README.md` for bare-metal experiment setup.
- `benchmarks/BENCHMARKS.md` for performance context.

## Asking for Help

When opening an issue, include:

- Your operating system.
- The command you ran.
- The DByte file or minimal source that reproduces the issue.
- Full terminal output.
- Whether the issue is in the language runtime, DByteOS userland, Kernel Lab, packaging, or documentation.

## Experimental Areas

Kernel Lab, IRQ, PIC, IDT, page fault, and QEMU workflows are experimental. Use a VM and keep real data away from test runs.
